# Reto
## Descripción
Can you convert the number 42 (base 10) to binary (base 2)?
## Solución
### Solucion 1
- Usando cyberchef
- https://gchq.github.io/CyberChef/#recipe=To_Base(2)&input=NDI
-  recipe: to base: 2 
- input: 42
### Solucion 2
- usando Python
```
neylar11-picoctf@webshell:~$ python
>>> >>> bin(42)
'0b101010'
```
picoCTF{101010}

- bin es para hacer binarios
